Chamber of Commerce Site Plan

Site Name

"Sauces City" Chamber of Commerce

Site Purpose

We know that there are so many businesses in the Sauces colony that it can be hard to find specific products and services. This chamber of commerce helps people to know more about the products and services provided in this colony so that they can easily find what they are looking for. Also, this site acts as a cheap marketing resource for businesses.

Scenarios #1

Where can I find a good place to eat?

Scenarios #2

I would like to know where can I buy perfumes and beauty & hygiene products.

Scenarios #3

What events will the Mexican Soccer Federation have in the next month?

Color Schema

Color 1: #FFFFFF

This color will be used for backgrounds, so, most of the page will be in this color.

Color 2: #383838

This color will be used for heading backgrounds, borders, and footer background.

Color 3: #E1BC4F

This color will be used for heading texts, nav and hover links.

Color 4: #871459

This color will be used for nav hover, active nav button and links.

Color 5: #000000

This color will be used for the body text and shadows.
